**AI Language Translation Tool**
This project was developed as part of the CodeAlpha Artificial Intelligence Internship.

Project Description:
The AI Language Translation Tool is a Python-based web application that translates text between different languages using AI-powered translation systems.

 Features:
- Translate text between multiple languages
- Simple user interface
- Fast translation
- Built using Streamlit

Technologies Used:
- Python
- Streamlit
- deep-translator

How to Run:
```bash
pip install streamlit deep-translator
streamlit run app.py
```
